Огляд
-----

Проект складається з 4 основних додатків:

accounts
    Автентифікація та управління користувачами

credits
    Кредитні заявки та прогнозування

analytics
    Генерація та відображення аналітичних графіків

api
    REST API для зовнішніх інтеграцій
